## 克隆仓库

karin根目录执行以下命令克隆仓库到本地

```bash
git clone https://github.com/karinjs/karin-plugin-chatgpt.git ./plugins/karin-plugin-chatgpt
```

## 安装依赖

```bash
pnpm install --filter=karin-plugin-chatgpt
```


## 开发调试

```bash
node . --dev
```
